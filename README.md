# toast
Din alfa og omega-guide til hvordan toaste

## Bare minimum
- Grovbrød
- Fint brød
- Jarlsberg
- Revet ost
- Smør
- Sopp
- Tomatsaus
- Soltørka tomat
- Siliana salami
- Løk

## Nais additions
- Truffel mayo
- Grov sennep
- Sprøstekt løk
- Sjokolademelk (gjerne Tine melk + Oboy)
- Pesto (funker fint med både tradisjonell "kun pinjekjerner", men også potetvariantene)
